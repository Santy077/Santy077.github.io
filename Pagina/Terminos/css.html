CSS

Lenguaje utilizado para describir la presentación visual de documentos HTML y XML. CSS define cómo se deben mostrar los elementos HTML en términos de diseño, formato, colores, fuentes y otros aspectos visuales. Permite separar el contenido estructural de una página web (definido en HTML) de su diseño y estilo (definido en CSS), lo cual facilita la creación de sitios web con un aspecto visual coherente y profesional.
