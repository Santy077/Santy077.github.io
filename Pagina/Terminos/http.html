HTTP

Es el protocolo estándar utilizado para la comunicación entre clientes (generalmente navegadores web) y servidores en la World Wide Web. HTTP define cómo se solicitan y transfieren recursos como páginas web, imágenes, videos y otros archivos entre el cliente y el servidor. Utiliza métodos de solicitud como GET y POST para enviar y recibir datos, estableciendo una conexión temporal entre el cliente y el servidor para cada solicitud realizada.
