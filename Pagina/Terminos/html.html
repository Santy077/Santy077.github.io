HTML

Lenguaje de marcado utilizado para crear y estructurar contenidos en páginas web. Define la estructura y el contenido de una página web mediante etiquetas o elementos que describen cómo debería ser presentado el contenido en un navegador web. HTML utiliza una sintaxis sencilla y permite la incorporación de otros recursos multimedia como imágenes, videos y hojas de estilo (CSS) para mejorar el diseño y la apariencia de las páginas web.
